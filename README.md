# 🚀 ProductivityVision ML (Beta 1.0)

**ProductivityVision** — це прогнозно-аналітична система для оцінки рівня продуктивності співробітників, розроблена в рамках кваліфікаційної бакалаврської роботи.

Проєкт використовує методи машинного навчання (**XGBoost**) для виявлення ключових факторів впливу на ефективність роботи та надання автоматизованих рекомендацій для HR-менеджерів.

---

## 📋 Основний функціонал

Система вирішує три головні задачі HR-аналітики:

1.  **📊 Моніторинг та Аналітика (Dashboard)**
    * Перегляд поточної бази співробітників з фільтрацією та сортуванням.
    * Візуалізація демографічних та професійних даних (розподіл за віком, стажем, департаментами).
    * Використання датасету *Remote Worker Productivity*.

2.  **🤖 Прогнозування (Prediction)**
    * Миттєвий розрахунок **Productivity Score** для нових кандидатів.
    * Введення параметрів через зручну форму (освіта, досвід, умови домашнього офісу тощо).

3.  **💡 Рекомендаційна система (ML Advice)**
    * **What-if аналіз:** Симуляція зміни умов праці.
    * Генерація конкретних порад (наприклад, *"Збільшити кількість днів віддаленої роботи на 2 дні"*).
    * Прогноз приросту ефективності у відсотках.

## 🛠 Технологічний стек

* **Мова:** Python 3.11+
* **Інтерфейс (UI):** CustomTkinter (Сучасний дизайн, Dark Mode).
* **Machine Learning:** XGBoost Regressor (Точність моделі $R^2 \approx 0.94$).
* **Data Processing:** Pandas, NumPy, Scikit-learn.
* **Visualization:** Matplotlib, Seaborn.
